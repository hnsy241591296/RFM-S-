# 🛒 基于RFM-S模型的社交电商用户分层与AI智能营销系统

> 🚀 **项目愿景**：将传统数据挖掘（聚类分析）与大语言模型（LLM）相结合，打通从“看懂数据”到“自动化业务触达”的完整闭环。

## 💡 项目背景与业务价值
在社交电商场景下，传统的 RFM 模型难以衡量用户的“社交裂变”价值。本项目在 RFM 基础上引入了 **S (Social/Share 社交互动维度)**，精准定位诸如“高价值社交达人”、“沉睡潜力客”等细分群体。

针对业务端“精细化运营文案撰写成本高”的痛点，本项目通过 Streamlit 搭建交互页面，并接入 AI 大模型，实现**输入用户 ID -> 识别分层标签 -> AI 一键生成个性化裂变文案**的自动化工作流。

## ✨ 核心功能演示 (Features)
*(提示：这里强烈建议以后补充一张你网页运行时的截图或录屏 GIF)*

1. **📊 动态用户画像提取**：输入目标用户 ID，即刻展示其 R、F、M、S 原始指标及最终归属的客群标签。
2. **🤖 AI 智能文案生成**：基于用户的分层画像，Prompt 驱动 LLM 自动生成 3 条针对性极强的营销/挽回文案。
3. **📈 深度数据可视化**：项目内置数据探索分析（EDA），包含 **PCA 降维聚类散点图、客群特征雷达图、箱线图**等，直观展示模型分层效果。

## 🛠️ 技术栈 (Tech Stack)
- **数据科学与算法**：Python, Pandas, Scikit-learn (K-Means 聚类, PCA 降维)
- **前端交互**：Streamlit (极速构建全栈数据 Web 应用)
- **数据可视化**：Matplotlib / Seaborn
- **大模型接入**：LLM API (Prompt Engineering)
